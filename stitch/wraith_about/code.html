Wraith is a high-security cryptographic tool designed for the unshakeable preservation of on-chain privacy. It operates as a digital safe, utilizing stealth addresses to decouple your identity from your transactions.

Mechanism 01

Stealth Addressing

Every time you receive assets via Wraith, a unique, one-time address is generated. To an outside observer, the transaction appears to be sent to a fresh wallet with no history. Only the intended recipient can calculate the private key for this address.

Unlike traditional mixers that pool funds, Wraith focuses on Recipient Privacy. By utilizing Diffie-Hellman key exchanges over the Elliptic Curve (secp256k1), Wraith allows a sender to create an obfuscated destination that only the receiver can claim.

This effectively breaks the link between the receiver's public identity and their actual on-chain holdings, creating a "Digital Vault" that remains invisible to public scrapers and analytic engines.

Mechanism 02

The ERC-5564 Standard

Wraith is built upon the rigorous technical foundations of ERC-5564. This standard defines a universal way for users to announce their stealth meta-addresses, allowing any compatible application to send them private payments without prior interaction.
